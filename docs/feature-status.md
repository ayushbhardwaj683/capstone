# Feature Status

## Completed Features

- Multi-tenant architecture with user-owned forms and collaborator-aware access checks
- Drag and drop form builder scaffold in the frontend builder surface
- Advanced field type schema for text, email, phone, number, select, date, rating, linear scale, file upload, and signature
- JWT authentication flow with register and login endpoints

## In-Progress Features

- Conditional logic engine for dynamic field visibility based on previous answers
- Analytics dashboard with response charts, trends, and breakdowns
- Public sharing links for secure access to published forms

## Suggested Next Implementation Order

1. Finish conditional logic evaluation in the public renderer and builder preview
2. Replace mock analytics with aggregated backend data
3. Add publish and unpublish UI with slug management and sharing controls
